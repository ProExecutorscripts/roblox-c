# roblox-c
C and C++ to Luau compiler for Roblox.

## Supports
- Enums
- If statements
- Return
- Functions
- Calling functions
- Binary operators
- Unary operators
- Preprocessor
- Struct
- do-while loops
- Strings
- elseif
- For loops
- casting
- else
- `*` & `&`
- Unions
- `main` functions
- `++` & `--`
- switch (with fallthrough)
