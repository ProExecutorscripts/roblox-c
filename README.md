# roblox-c
C and C++ to Luau compiler for Roblox.

## Supports
### C with:
- Enums
- If statements
- Return
- Functions
- Calling functions
- Binary operators
- Unary operators
- Preprocessor
- Struct
- do-while loops
- Strings
- elseif
- For loops
- casting
- else
- `*` & `&`
- Unions
- `main` functions
- Gotos (Lua 5.2+)
- Short & Long values
- `++` & `--`
- Arrays
- Indexing
- switch (with fallthrough)
### C++ with:
- everything in C
- classes
- constructors
- delete
- new
- namespaces
- destructors
